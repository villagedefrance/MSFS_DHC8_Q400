DE HAVILLAND CANADA DHC-8 Q400 PROCEDURES

For detailed instructions on how to fly this aircraft, see the Aircraft Information
articles in the Learning Center. For suggested speeds, see the Reference page of the Kneeboard. Note that most actions can also be performed using the mouse.
 

 

BEFORE START  
   
[ ] Parking Brake SET
[ ] Fuel Quantity CHECK
[ ] Engine Start Levers CHECK IDLE
[ ] Propeller Controls Levers CHECK FULL FORWARD
   

PUSHBACK (if parked at a gate)

 

  

[ ] Pushback


 

REQUEST
(press SHIFT+P, then 1 for tail-left or 2 for tail-right, then press SHIFT+P to stop)
   
ENGINE START (AUTOSTART)  
   
Press CTRL+E to initiate engine autostart sequence.

AFTER START  
   
[ ] Avionics Master Switch ON
[ ] Pitot Heat ON
[ ] De-ice AS REQUIRED
[ ] Engine Start Levers CHECK IDLE
[ ] Propeller Controls Levers CHECK FULL FORWARD
[ ] Autofeather ARM
[ ] Lights AS REQUIRED
[ ] Autopilot SET and OFF
[ ] Instruments CHECKED
[ ] Trim Tabs SET FOR CLIMB
[ ] Seatbelt Sign ON
[ ] No Smoking Sign ON
   
DURING TAXI / BEFORE TAKEOFF
   
[ ] Spoilers (PFCS) SET TO TAXI
[ ] Brakes RELEASE
[ ] Thrust Levers INCREASE SLIGHTLY TO ROLL
[ ] Thrust Levers IDLE FOR TAXI
[ ] Flight Controls CHECK
[ ] Flaps SET FOR TAKEOFF (5 or 10)
[ ] Brakes HOLD
   
CLEARED FOR TAKEOFF  
   
[ ] Engine Start Switches GEN
[ ] Flight Director ON
[ ] Spoilers (PFCS) SET TO FLIGHT
[ ] Transponder AS REQUIRED
[ ] Lights AS REQUIRED
   
TAKEOFF  
   
[ ] Brakes RELEASE
[ ] Thrust Levers ADVANCE SMOOTHLY TO 40% N1
   
--WAIT FOR ENGINE STABILIZATION--
 
[ ] Thrust Levers ADVANCE SMOOTHLY TO 100% N1
[ ] Propeller Controls Levers CHECK FULL FORWARD
[ ] Engine Instruments MONITOR
[ ] Airspeed 80 KIAS CALLOUT "80 KNOTS"
   
-- VERIFY PROPER THRUST SET--
   
[ ] Airspeed V1 CALLOUT "V1"
[ ] Airspeed VR CALLOUT "ROTATE"
   
--ROTATE TO APPROX. 8 DEGREES PITCH UP--
   
[ ] Airspeed V2 CALLOUT "V2"
[ ] Positive Climb CALLOUT "POSITIVE CLIMB"
[ ] Landing Gear CALLOUT "GEAR UP"
[ ] Landing Gear UP AND LOCKED
[ ] Airspeed MAINTAIN V2+20 KIAS
[ ] Flaps UP AT 175 KIAS
[ ] Thrust Levers 90% N1
[ ] Airspeed 220 KIAS AT 3,000' AGL
[ ] Landing Gear Lever Verify OFF
   
CLIMB  
   
[ ] Airspeed MAINTAIN 250 KIAS
[ ] Landing Lights OFF ABOVE 10,000' MSL
[ ] Airspeed 280 KIAS ABOVE 10,000' MSL
[ ] Altimeter
  		SET TO 29.92 CROSSING 18,000' MSL
   
CRUISE  
   
[ ] Thrust Levers AS DESIRED
[ ] Propeller Controls Levers AS REQUIRED
[ ] Trim AS NECESSARY

  		SET TO LOCAL SETTING CROSSING 18,000' MSL

APPROACH

[ ] Flaps AS DESIRED

[ ] Thrust Levers AS DESIRED
[ ] Propeller Controls Levers AS REQUIRED
[ ] Landing Gear DOWN AND CONFIRMED
[ ] Flaps AS DESIRED
[ ] Autopilot AS DESIRED
   
LANDING ROLL

[ ] Autopilot CHECK DISENGAGED
[ ] Speedbrake Lever CHECK FULL UP
[ ] Brake AS NECESSARY
[ ] Thrust Levers REVERSE
[ ] Thrust Levers IDLE AT 60 KIAS
